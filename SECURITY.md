# Security Policy

## Supported Versions

Security fixes are handled for the latest code on the `main` branch and the latest published release.

## Reporting a Vulnerability

Please do not report security vulnerabilities in a public issue.

Use GitHub's private vulnerability reporting or open a private security advisory for this repository. Include:

- A clear description of the issue
- Steps to reproduce it
- The affected version or commit
- Any relevant screenshots, logs, or proof of concept details

This project is a local Windows desktop app. Useful security reports are most likely to involve installer behavior, local file handling, startup registration, screenshot capture behavior, or unintended exposure of user data.
